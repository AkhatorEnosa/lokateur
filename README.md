# Lokateur 📍

**Lokateur** is a modern location-based platform designed to simplify the process of finding a location based on IP Address, Lokateur provides a seamless, map-integrated experience.

## 🚀 Features

* **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Tech Stack

* **Frontend**: React.js & TailwindCSS
* **Maps API**: Leaflet

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

* [Node.js](https://nodejs.org/) (v14.x or higher)
* [NPM](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

## ⚙️ Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/AkhatorEnosa/lokateur.git
cd lokateur

```


2. **Install dependencies**
```bash
npm install

```


4. **Run the application**
```bash
npm run dev

```


The app should now be running on `http://localhost:5173`.
